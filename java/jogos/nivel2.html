Variáveis e Tipos em Java

O que são Variáveis?

Variáveis são espaços na memória que guardam dados. Pense nelas como caixas onde você pode guardar diferentes tipos de informações.

Tipos Principais:

int

Números inteiros

int idade = 25;

double

Números decimais

double altura = 1.75;

String

Texto

String nome = "João";

boolean

Verdadeiro/Falso

boolean ativo = true;

Regras Importantes:

  • Variáveis precisam ser declaradas antes de usar
  • O nome não pode começar com número
  • Cada tipo guarda valores específicos
  • Use nomes que façam sentido

Exemplo Prático:

String nomeProduto = "Notebook";
double preco = 2599.99;
int quantidade = 5;
boolean disponivel = true;
